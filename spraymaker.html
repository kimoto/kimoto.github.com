spraymaker

Easy to create high quality VTF file for Valve SourceEngine Games.
You can drag & drop any image files to spraymaker.exe

English / Japanese

Description

  • Output 256 x 256 size spray
  • Output alpha channel VTF file(VXT5 = high quality compression type)
  • Add high quality VTF option(NOMAP, NOLOD)
  • Read can 200 over image types
  • Keep original image aspect ratio(auto padding transparent pixel)
  • Read can directory (recursive convertion)
